# Vue Мессенджер

Простой мессенджер на Vue 3 с функционалом:
- Регистрация и авторизация
- Поиск пользователей по email или имени
- Переписка в реальном времени через WebSocket
- Просмотр и удаление профиля

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

- `src/views/` - страницы приложения (Login, Register, Home, Chat, Profile)
- `src/services/` - API сервисы и WebSocket клиент
- `src/router/` - настройка роутинга

## API

Приложение работает с бэкендом FastAPI, который должен быть запущен на `http://localhost:8000`
